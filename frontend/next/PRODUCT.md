# Product: TrainWithGouli

## Vision
A no-nonsense, brutalist workout tracker that exposes structure and rewards consistency.

## Audience
Strength-training enthusiasts who want a minimal, fast logbook without marketing fluff.

## Key principles
- Raw, unpolished honesty.
- High contrast, system/monospace typography, exposed borders.
- Every action is deliberate; no decorative gradients or soft shadows.
- Data ownership through Supabase RLS.

## Features
1. Auth (sign up / log in / log out).
2. Exercise library with YouTube embeds.
3. Workout plan builder.
4. Daily workout logging with weight nudge.
5. Read-only social feed of workout days.

## Roles
- `user`: create/read exercises, plans, days; edit/delete own content.
- `admin`: edit/delete any content.
